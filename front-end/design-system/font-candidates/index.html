An open-source font for reef-pi

Five geometric sans candidates, each shown with body copy and a Bento OS tile readout. Picked against Century Gothic DNA, weight range needs, and Pi-over-LAN bundle size.

Recommendation

Manrope + JetBrains Mono

Manrope keeps the geometric, round Century Gothic character that defines the reef-pi voice — but supplies the 400 / 500 / 600 / 700 range Bento OS tiles depend on. JetBrains Mono pairs naturally for the big monospace numerics (gauges, sparkline readouts, threshold values). Both are SIL OFL, both load fast.
