RangeSelector

Segmented radio control for time-range selection. Persists to localStorage with 250ms debounce.

Default — value="1d"
Time range

options=['1h','6h','1d','7d','30d']

Compact — labels trimmed to digits
Time range

compact — "1H" → "1"

Keyboard — tab in, arrow keys
Time range

Tab → focus ring. ←/→ to step. Enter/Space to select.

Custom options — scope="dashboard"
Dashboard time range

options=['1h','6h','1d'] scope="dashboard"
